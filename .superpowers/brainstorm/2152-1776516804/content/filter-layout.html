A — 顶部 Tab 筛选栏

分类 Tab 横排在卡片网格上方，现货/可调货切换放右侧。布局最简洁，内容区最宽，适合移动端。

优点

  • 实现最简单
  • 内容区宽，可3列
  • 移动端友好

缺点

  • 多条件同时筛选弱
  • Tab过多时会换行

B — 左侧固定侧边栏

左侧竖排筛选面板，显示各类别数量，右侧 2 列卡片网格。风格类似 B2B 采购平台，适合多条件筛选。

优点

  • 多条件并行筛选
  • 显示每类数量直观
  • 可扩展更多筛选维度

缺点

  • 内容区变窄(2列)
  • 移动端需抽屉处理

C — Pill Chip 横排（推荐）

圆角 pill 样式横排分类标签，右侧"现货优先"开关 + 当前结果数量。3 列卡片网格。风格最接近 Linear/Vercel，克制且现代。

优点

  • 视觉最轻量
  • 3列卡片，信息量最大
  • 与整体深色风格契合度高

缺点

  • 仅支持单选分类
  • 类别超多时需折行
